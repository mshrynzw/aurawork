## 結合テストとは

結合テスト（Integration Test）は、複数のコンポーネントの連携をテストするものです。

### 特徴

- データベースを使用する
- リレーションシップをテスト
- 複数のコンポーネントが連携して動作することを確認
- ビジネスロジックの統合をテスト

### 対象

- モデル間のリレーション
- 複数テーブルの連携
- ビジネスロジックの統合

---

## テストの実行

```bash
# すべての結合テストを実行
php artisan test tests/Integration

# 特定のテストファイルを実行
php artisan test tests/Integration/PayrollCalculationIntegrationTest.php

# 詳細表示
php artisan test -v
```
