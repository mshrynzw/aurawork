## 単体テストとは

単体テスト（Unit Test）は、個別のメソッドや関数をテストするものです。

### 特徴

- 外部依存がない（データベース、API等をモック）
- 高速に実行できる
- リソースをあまり使わない
- デバッグが容易

### 対象

- モデルのCRUD操作
- 計算ロジック
- バリデーション
- ユーティリティ関数

---

## テストの実行

```bash
# すべての単体テストを実行
php artisan test tests/Unit

# 特定のテストファイルを実行
php artisan test tests/Unit/AccountInfosTest.php

# 特定のメソッドを実行
php artisan test --filter test_account_info_creation
```
