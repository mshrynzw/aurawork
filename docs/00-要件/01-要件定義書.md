# 要件定義書（aurawork）
作成日: 2025-11-12

## 1. 概要
Aurawork は、勤怠・休暇・給与・承認を統合管理する人事労務プラットフォーム。マルチテナント対応。

## 2. 技術構成
Laravel 11 / React 18 + Vite / Tailwind CSS + shadcn/ui / MySQL 8 / Sanctum 認証 / Vercel・Render

## 3. 機能要件
- マスタ（m_）: 会社・部門・ユーザー・雇用情報・ロール/権限・賃金項目・休日・承認フロー
- 勤怠/休暇（t_）: 予定・打刻・申請・残高
- 給与（t_）: 締め・明細・明細行
- 承認（m_/t_/h_）: フロー定義・紐づけ・履歴

## 4. 非機能
セキュリティ、監査、性能、運用、テスト方針を定義
